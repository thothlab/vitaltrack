# VitalTrack Bot — Полное ТЗ (Full Specification)

## TL;DR
Это полноценная платформа медицинского мониторинга в Telegram:
- пациент ведёт дневник
- врач наблюдает
- есть отчёты, напоминания, калькуляторы

---

# 1. Концепция
Универсальный терапевтический дневник пациента с поддержкой:
- давления
- сахара
- лекарств
- симптомов
- питания
- отчётов
- связи с врачом

---

# 2. Роли

## Пациент
- ввод данных
- просмотр истории
- отчёты
- связь с врачом

## Врач
- список пациентов
- просмотр данных
- сообщения
- alerts

---

# 3. Ключевые принципы

- всегда запрашивать время события
- поддержка ввода “задним числом”
- несколько записей в день
- минимальный ввод текста
- кнопки + guided UX

---

# 4. Модули

## Давление
- САД / ДАД
- пульс
- сессии измерений
- несколько измерений

## Сахар
- значение
- контекст (до/после еды)
- время

## Лекарства
- точное время приёма
- напоминания
- adherence

## Симптомы
- самочувствие (градации)
- список симптомов
- интенсивность

## Питание
- приёмы пищи
- теги

## Анализы
- холестерин
- ЛПНП/ЛПВП
- триглицериды
